# Task 구현 프롬프트 템플릿

```
Task [ID]: [Task명] 구현

구현 요구사항:
- TDD 방식으로 테스트 케이스 먼저 작성
- SDD 원칙에 따른 설계 문서 기반 구현
- 클린 코드 원칙 준수
- 에러 핸들링 및 로깅 포함

구현 순서:
1. 테스트 케이스 작성
2. 최소 기능 구현 (Red-Green-Refactor)
3. 리팩토링 및 최적화
4. 문서 업데이트

산출물:
- 소스 코드
- 단위 테스트
- API 문서 (해당시)
- 사용자 가이드 (해당시)

검증 기준:
- 모든 테스트 통과
- 코드 커버리지 80% 이상
- 성능 요구사항 충족
```
